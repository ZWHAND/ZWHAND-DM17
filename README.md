2、Host 
    Computer‌: Includes the host computer software and user manual for the "ZWHAND-DM17" dexterous hand.
    
3、ROS‌: 
    Contains the ROS1 and ROS2 packages for the "ZWHAND-DM17" dexterous hand.
    
4、URDF‌:
